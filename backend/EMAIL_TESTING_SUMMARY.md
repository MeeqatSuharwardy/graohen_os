**What Needs Setup**:
1. **DNS Configuration** (Cloudflare):
   - A record: `smtp.fxmail.ai` → Server IP
   - A record: `mail.fxmail.ai` → Server IP
   - MX record: `fxmail.ai` → `mail.fxmail.ai`
   - SPF record: `v=spf1 ip4:YOUR_SERVER_IP -all`
   - DKIM record: Generated by Postfix
   - DMARC record: `v=DMARC1; p=none`

2. **Postfix Installation**:
   - Install Postfix on server
   - Configure Postfix to use SMTP authentication
   - Set up TLS certificates
   - Configure email ingestion script

3. **Environment Variables**:
   ```bash
   SMTP_HOST=smtp.fxmail.ai  # Or use mail.fxmail.ai
   SMTP_PORT=587
   SMTP_USERNAME=postmaster@fxmail.ai
   SMTP_PASSWORD=STRONG_PASSWORD
   SMTP_USE_TLS=true
   ```

## Testing Commands

### Test Email Send API
```bash
# Get auth token
TOKEN=$(curl -X POST http://localhost:8000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"pass"}' \
  | jq -r '.access_token')

# Send email
curl -X POST http://localhost:8000/api/v1/email/send \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
```
